What Is BaitLogic?
BaitLogic is a free local fishing intelligence platform built specifically for Southern Illinois and St. Louis Metro anglers. No paywalls. No subscriptions. No gatekeeping.
Built by a self-taught angler from Highland/Collinsville, IL — on a phone, with no outside funding — because the tools we needed never existed.
Features
🌡️ Live Conditions — Real-time air and water temps via NOAA and USGS
📊 Depth Finder — Free depth charts for Carlyle Lake, Silver Lake, Horseshoe Lake, and the Mississippi River
👥 Community Reports — Real bite reports from real local anglers
🔬 Waterway Health Index — Citizen science water quality tracking and Guardian program
🧹 Clean Up Crew — Community cleanup events protecting Carlyle Lake
🐠 Species ID — Visual guides for local species and seasonal patterns
🎣 Catch Log — Personal log that works offline
🛒 Marketplace — Local gear swap and trade
📡 Business Hub — Tools for local bait shops, marinas, and guides
Tech Stack
Pure HTML / CSS / JavaScript — single file
GitHub Pages — free hosting
Formspree — form backend
NOAA API — live weather
USGS API — live water temperature
Service Worker — offline Deep Woods protocol
localStorage — catch log and crew signups
Mission
Three pillars drive everything:
Community First — Free forever for the bank fisher, the beginner, and the underserved angler who never had the right tools.
Real Intelligence — Live government data, community reports, and depth charts in one place before you leave the house.
Conservation Always — The Waterway Health Index and Clean Up Crew exist to protect Carlyle Lake and Silver Lake for every angler who comes after us.
